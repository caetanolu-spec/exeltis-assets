# Exeltis Assets

Repositório público estático para imagens de produtos Exeltis usadas no Bling.

## Estrutura

- `produtos/`: imagens extraídas e nomeadas por SKU.

## GitHub Pages

Após publicar este repositório como público no GitHub e habilitar GitHub Pages a partir da branch principal, as imagens devem ficar disponíveis em:

```text
https://caetanolu-spec.github.io/exeltis-assets/produtos/NOME_DO_ARQUIVO.png
```

## Bling

Usar o arquivo `bling_atualizacao_imagens_github_pages.csv` gerado em `C:\Users\caeta\Downloads\exeltis-bling\imagens` para atualizar os produtos por planilha.

Observação: o Bling exige URLs públicas HTTPS. Antes de importar, testar 2 ou 3 URLs no navegador depois que o GitHub Pages estiver ativo.
